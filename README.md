#  Drone Forensics Digital Twin  
### Anomaly Detection • Telemetry Analysis • LSTM Autoencoder • PX4 SITL

A digital twin system for drone forensic analysis, enabling real-time telemetry ingestion, simulation, anomaly detection using LSTM Autoencoders, and automated forensic reporting.

---

## Features

- **Digital Twin Simulation** using PX4 SITL / Gazebo  
- **Real-Time Telemetry Ingestion** (PX4 logs, CSV, MAVLink-derived data)  
- **LSTM Autoencoder–based Anomaly Detection** with reconstruction-error scoring  
- **Interactive Dashboard** for visualizing flight behavior, anomalies, and timelines  
- **Automated Forensic Reports** in JSON/PDF  
- **Modular Pipeline:** ingestion → preprocessing → model → reporting
